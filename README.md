# Қыз ұзату — шақыру сайты

Статикалық сайт (HTML/CSS/JS). GitHub Pages-те тегін орналасады.

## 📝 Деректерді өзгерту

Барлық мәтін, күн, аты, мекенжай — `script.js` файлының басындағы **CONFIG** блогында.
Тек соны өзгерт, қалғанын тиіспе.

- `heroName` — қыздың аты
- `eventDateTime` — той күні/уақыты (кері санақ осыдан есептеледі), формат `2026-08-15 17:00:00`
- `venueName`, `venueAddress`, `mapUrl` — мекенжай мен карта сілтемесі
- `sheetUrl` — RSVP жауаптары жазылатын Google-кесте сілтемесі
  (баптауы: **GOOGLE-SHEET-SETUP.md** файлын қара)

## 🖼 Фотоны ауыстыру

`images/` папкасында екі сурет:
- **`bg.jpg`** — басты экранның гүлді фоны
- **`saukele.jpg`** — ортадағы сәукеле суреті

Ауыстырғың келсе — сол атаулармен жаңа суретті үстіне көшір (атауын өзгертпе).

## 🚀 GitHub Pages-ке жариялау

1. GitHub-та жаңа repository аш (мыс. `adiya-kyz-uzatu`), **Public**.
2. Осы папканың барлық файлын сол репозиторийге жүкте (git немесе сайттан "upload files").
3. Repository → **Settings → Pages**.
4. **Source**: `Deploy from a branch`, Branch: `main`, папка: `/ (root)` → Save.
5. 1–2 минуттан соң сайт мына мекенжайда ашылады:
   `https://СЕНІҢ-АТЫҢ.github.io/РЕПОЗИТОРИЙ-АТЫ/`

## 🔧 Жергілікті тексеру (компьютерде)

Файлдарды браузерде ашсаң жеткілікті: `index.html` екі рет бас.
Не болмаса жергілікті сервер:

```powershell
python -m http.server 8000
# браузерде: http://localhost:8000
```
