# Запись ответов RSVP в Google-таблицу — инструкция

Сайт статический, поэтому ответы принимает бесплатный «сервер» — Google Apps Script.
Настройка ~5 минут. Пункты меню даны как в русском интерфейсе Google Таблиц.

## 1. Создай таблицу
1. Открой https://sheets.new — создастся новая таблица.
2. Назови её, например **«Той RSVP»**. Заголовки писать не нужно —
   скрипт сам добавит строку `Уақыты | Аты-жөні | Жауабы`.

## 2. Вставь скрипт
1. В таблице сверху: **Расширения → Apps Script**.
2. В открывшемся окне удали весь код, который там был, и вставь код
   из файла `google-apps-script.gs` (целиком).
3. Сохрани: значок 💾 или **Ctrl+S**.

## 3. Опубликуй как веб-приложение
1. Справа сверху: **Развернуть → Новое развертывание**
   (англ. Deploy → New deployment).
2. Нажми ⚙️ рядом с «Выберите тип» → выбери **Веб-приложение** (Web app).
3. Параметры:
   - **Описание**: той rsvp (любое)
   - **Запуск от имени** (Execute as): **Я** (Me)
   - **У кого есть доступ** (Who has access): **Все** (Anyone)  ← важно!
4. Нажми **Развернуть** (Deploy).
5. Google попросит доступ → выбери свой аккаунт →
   «Дополнительные настройки» → «Перейти на страницу … (небезопасно)» →
   **Разрешить**. Это нормально — скрипт пишет только в твою таблицу.
6. Скопируй **URL веб-приложения**. Он выглядит так:
   `https://script.google.com/macros/s/AKfyc.../exec`

## 4. Вставь ссылку в сайт
1. Открой `script.js`.
2. В блоке CONFIG найди:
   ```
   sheetUrl: "ОСЫ_ЖЕРГЕ_APPS_SCRIPT_СІЛТЕМЕСІН_ҚОЙ",
   ```
3. Замени текст в кавычках на URL из шага 3. Сохрани.

## 5. Проверь
Открой сайт, заполни форму, нажми **Жауапты жіберу**.
В таблице должна появиться новая строка. ✅

---
### Если строка не появляется
- Проверь «У кого есть доступ» = **Все** (Anyone).
- `sheetUrl` должен заканчиваться на `/exec` (не `/dev`).
- Если менял код — переразверни: **Развернуть → Управление развертываниями →
  «карандаш» (изменить) → Версия: Новая версия → Развернуть**.
